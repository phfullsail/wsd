<h1>
Main heading element.
<h2>
Sub heading level 2.
<h3>
Sub heading level 3.
<h4>
Sub heading level 4.
<h5>
Sub heading level 5.
<h6>
Sub heading level 6.
<p>
Used for creating a paragraph.
<b>
Creates bold text.
<i>
Creates italic text.
<sup>
Used for creating superscript characters.
<sub>
Used for creating subscript characters.
<br>
Adds a line break.
<hr>
Adds a horizontal rule to create a break between themes.
<strong>
Used to show that content has strong importance.
<em>
Used emphasize content.
<blockquote>
Used for long quotes, usually at least a paragraph.
<q>
Used for short quotes.
<abbr>
Used for abbreviations or acronyms.
<cite>
Used for creating citations.
<dfn>
Used for defining a new term.
<ol>
Used to create an ordered list.
<li>
Adds a new item to list elements.
<ul>
Used to create an unordered list.
<dd>
Used to create a definition list.
<dt>
Defines a new term for the definition list.
<dd>
Element used for the definition in a definition list.
